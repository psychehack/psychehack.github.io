浅析奥佳华的ROE

发表于 2019-06-12 | 分类于 金融交易

ROE（净资产收益率）是认识了解企业生意特征，判断其目前所处发展周期的重要视角，由于该指标较为综合地展现了企业运营中的几个关键因素，从而使得它成为研判企业经营发展的重要工具。本文将尝试从ROE杜邦分解的角度对奥佳华（002614.SZ）的经营特征做出分析。

ROE杜邦分析主要从资金杠杆、利润率以及周转率等三个方面展开。

首先，从杠杆的角度而言，奥佳华2018年的资产负债率达到44.6%，权益乘数1.76，继续加杠杆的潜力有限。

就总资产周转率而言，公司2013年确认拐点，之后六年总资产周转率随着销售规模的逐步扩大，总体保持了上升态势。不过，由于有新的在建工程和固定资产投入，奥佳华的总资产周转率短期走高的可能性不高。

奥佳华固定资产占总资产的比重仅为10.52%，流动资产中的现金比例较高31.6%，资产运营方面的存货、应收账款、流动资产的周转速度都较快，具有轻资产运营的一些特征，主要原因还是因为流动资产占总资产比重较高。

归根结底，奥佳华终究还是制造业企业，生意模式属于资本推动型。虽然近几年公司一直通过订单外包、租赁厂房的方式来扩张产能，但该方式的不可控性以及不可持续性还是在客观上要求公司只有通过上马新的生产线和产能的扩张，才能实现销售规模和收入的增长。一个明证是，公司总资产从2010年底的10.55亿扩张到2018年底的59.61亿，增长了465%，同期的销售收入从18.3亿增长到54.47亿，增长197%，固定资产从1.94亿涨到6.27亿，增长223%。

公司在互动平台上表示，2018年公司一直通过订单外包、租赁厂房等方式来弥补产能，因为市场需求强劲，公司决定在厦门和漳州开建两个智能园区，以彻底解决产能问题，新工厂2019年开建，2020年开始有新产能释放，2023年彻底达产，总投资13亿+，施工期5年，平均每年则有2.6亿的在建工程转为固定资产。

所以，短期看2019年、2020年，奥佳华的新工厂还处于建设期，产能部分释放阻滞销售收入提升，总资产周转率提升难度较大。不过中长期来看，只要市场保持对按摩椅、按摩小器具的需求，新厂稼动率提升到较高水平的话，总资产周转率会有一定的上升余地。

最后是净利润率，根据公式，毛利率 - 三项费用率 - 税率 = 净利润率。先看毛利率，2012年以来，奥佳华销售毛利率一直稳中有升，主要原因是技术领先带来产品结构升级（毛利率更高的按摩椅收入占比提升）以及准确的品牌营销推广策略（陈奕迅代言、赞助中国新歌声）品牌认知度和美誉度不断提升，进而带来平均单价的上行。

按摩椅和自有品牌是奥佳华毛利率提升的驱动力之所在，二者成长拉动奥佳华业绩的趋势也并没改变。公司三项费用率支出较稳定，一直在28%-29%之间波动。2017开始加大在渠道和营销上的投入，目前公司处于品牌建设期，预计不会减少销售费用，未来随着销售规模的提升，销售费用边际成本有望降低。此外，三费支出中企业税金或有惊喜，2019年，奥佳华适用的增值税税率由16%下降至13%，预计可少缴增值税1200万元，个人所得税960万元和印花税、城建税、教育费附加等约250万元，增厚净利润2410万。

未融资上市前，奥佳华的净资产收益率到过43%的高水平，在净利润率（7%-8%）较低的情况下，高ROE的维持主要是靠高杠杆（一度接近3的权益乘数）和超强的总资产周转率（接近1.9）。近几年，随着自有品牌的扩张，公司总的净利润水平已经接近或超过了上市前，在资金杠杆空间不大的情况下，做高ROE的关键在于净利润率的提升，具体的逻辑链条是这样的↓：

提高ROE的关键是→ 提升净利润率 → 销售费用边际成本降低 → 销售规模提升

提高ROE的关键是→ 提升净利润率 → 提高毛利率 → 依靠规模效应降低营业成本 → 销售规模提升

提高ROE的关键是→ 提升净利润率 → 提高毛利率 → 提高按摩椅、自有品牌产品在营收中的占比

提高ROE的关键是→ 提升净利润率 → 降低税费

总体来看，奥佳华的生意特征更接近资本推动型，也就是说其盈利主要来源于资产规模的不断扩张，而并非单位净资产的高收益率。2018年，公司净资产收益率已经达到14.31%的优良水平，未来1-3年内的在建工程、固定资产的增加将使得总资产周转率承压，ROE上行承受阻力。不过，中长期来看的话，公司在产能充分释放拉动收入规模增加后，反而有可能拉动总资产周转率提高，从而带动ROE提升。

因此，对奥佳华来说，未来的两个具有决定意义的观测点是：

  1. 公司的销售规模能否一直稳健提升
  2. 行业所处市场的空间究竟有多大，按摩椅的渗透率是否可以一直提升
奥佳华做的是一门什么样的生意？
