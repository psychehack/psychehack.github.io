浅析奥佳华的ROE

发表于 2019-06-12 | 分类于 金融交易

ROE（净资产收益率）是认识了解企业生意特征，判断其目前所处发展周期的重要视角，由于该指标较为综合地展现了企业运营中的几个关键因素，从而使得它成为研判企业经营发展的重要工具。本文将尝试从ROE杜邦分解的角度对奥佳华（002614.SZ）的经营特征做出分析。

奥佳华上市前，ROE最高曾达到52%，最低也有24%。2011年上市当年，ROE从43%陡然下滑到只有11%，次年2012年甚至只有1.37%。这很正常，因为ROE=净利润÷净资产，上市融资扩大了分母，而这部分资本又不能马上就转成收益增加净利润，所以公司上市当年一般都会经历ROE下滑这个过程。

2018年，奥佳华的ROE是14.3%，未来有无可能再做提高？ 我们从权益乘数、总资产周转率、净利润率三个方面来做一番探究。

从杠杆角度而言，奥佳华2018年的资产负债率达到44.6%，权益乘数1.76，继续加杠杆的潜力已有限。

总资产周转率呢？它提升的关键取决于公司的销售收入能否放大，并且在这个过程中不再继续从外部融资。奥佳华的总资产周转率在2013年确认拐点，之后六年随着销售规模的逐步扩大，总体保持了上升态势，这期间没有扩产和融资。不过，由于有新的一个多亿的在建工程和固定资产投入，加上12个亿的可转债项目，奥佳华的总资产周转率短期走高的可能性也不高。

奥佳华是制造业企业，生意模式属于资本推动型，一个明证是，公司总资产从2010年底的10.55亿扩张到2018年底的59.61亿，增长了465%，同期的销售收入从18.3亿增长到54.47亿，增长197%，固定资产从1.94亿涨到6.27亿，增长223%。

虽然近几年公司一直通过订单外包、租赁厂房的方式来扩张产能，但该方式的不可控性以及不可持续性还是在客观上要求公司只有通过上马新的生产线和产能的扩张，才能实现销售规模和收入的增长。尽管，它的固定资产占总资产的比重仅为10.52%，但这主要是因为流动资产占总资产比重较高造成的。

公司在互动平台上表示，2018年公司一直通过订单外包、租赁厂房等方式来弥补产能，因为市场需求强劲，公司决定在厦门和漳州开建两个智能园区，以彻底解决产能问题，新工厂2019年开建，2020年开始有新产能释放，2023年彻底达产，总投资13亿+，施工期5年，平均每年则有2.6亿的在建工程转为固定资产。

所以，短期看2019年、2020年，奥佳华的新工厂还处于建设期，产能部分释放阻滞销售收入提升，总资产周转率提升难度较大。不过中长期来看，按摩椅、按摩小器具在国内的渗透率提升，新厂稼动率提升到较高水平的话，总资产周转率未来会有一定的上升余地。

最后是净利润率，根据公式，毛利率 - 三项费用率 - 税率 = 净利润率。先看毛利率，2012年以来，奥佳华销售毛利率一直稳中有升，主要原因是技术领先带来产品结构升级（毛利率更高的按摩椅收入占比提升）以及准确的品牌营销推广策略（陈奕迅代言、赞助中国新歌声）品牌认知度和美誉度不断提升，进而带来平均单价的上行。

按摩椅和自有品牌是奥佳华毛利率提升的驱动力之所在，二者成长拉动奥佳华业绩的趋势也并没改变。公司三项费用率支出较稳定，一直在28%-29%之间波动。2017开始加大在渠道和营销上的投入，目前公司处于品牌建设期，预计不会减少销售费用，未来随着销售规模的提升，销售费用边际成本有望降低。此外，三费支出中企业税金或有惊喜，2019年，奥佳华适用的增值税税率由16%下降至13%，预计可少缴增值税1200万元，个人所得税960万元和印花税、城建税、教育费附加等约250万元，增厚净利润2410万。

未融资上市前，奥佳华的净资产收益率到过43%的高水平，在净利润率（7%-8%）不高的情况下，未来高ROE的维持主要是靠高杠杆（一度接近3的权益乘数）和超强的总资产周转率（接近1.9）。近几年，随着自有品牌的扩张，公司总的净利润水平已经接近或超过了上市前，在资金杠杆提升空间不大的前提下，做高ROE的关键在于净利润率的提升，具体的逻辑链条是这样的↓：

提高ROE的关键是→ 提升净利润率 → 销售费用边际成本降低 → 销售规模提升

提高ROE的关键是→ 提升净利润率 → 提高毛利率 → 依靠规模效应降低营业成本 → 销售规模提升

提高ROE的关键是→ 提升净利润率 → 提高毛利率 → 提高按摩椅、自有品牌产品在营收中的占比

提高ROE的关键是→ 提升净利润率 → 降低税费

一家制造业公司的ROE指标，通常会经历这样的一个过程：

拿到融资 → 建新的产线、厂房 → 分母（净资产或所有者权益）变大 → ROE变小 → 开始转化收益 → ROE重新上升到原有水平。

如果这家公司又同时满足以下条件：

  1. 过去2年的ROE保持在一个相对较高的水平。
  2. 目前刚融资（包括利润留存的内部融资）扩产，ROE下降到一个中等的水平，预计新的产品未来完全不愁销路。
  3. 公司是一个大行业中的小公司，过去增速较快，在行业中属于龙头地位，但占有率仅不到15%。

像这样的股票，我们就要注意了，它有成为长线大牛股的潜质。

问题是，奥佳华有吗？

要回答这个问题，关键就在于要搞清楚：

1、按摩椅、按摩器具这个行业潜在空间到底大不大（长长的雪坡）？

2、奥佳华在其所在的领域中是否有足够的竞争优势（以保证高毛利率）？

3、公司在成长的过程中是否需要大量的资本支出（不断融资）？

综合考虑以上的问题后，我认为奥佳华这家公司的前景还是光明的，未来应该可以走更远，ROE升到15%以上的概率比较大。不过，这并不意味着奥佳华属于高价值的企业，由于生意靠资本驱动，需要不断资本支出，留存现金的能力有所欠缺。同时，行业本身的缘故造成公司R特征不突出，净利润率可能永远无法突破10%，使得ROE的天花板较低（小于20%）。
