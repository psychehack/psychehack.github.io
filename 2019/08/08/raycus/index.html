锐科激光财务分析

发表于 2019-08-08 | 分类于 金融交易

看完锐科激光的生意特性表，最让人惊诧的就是净利润率了。一般情况下，一个制造业企业的净利润率能达到30%是非常罕见的，这种水平甚至超过许多软件公司，逼近一些白酒企业。从研究公司的角度讲，弄清楚为什么锐科能享有如此之高的净利润率，以及判断这种净利润率是否具有可持续性，显然是一件非常重要的事情。这是第一个问题。

公司成长性非常好，业务的快速扩张带来了超高速成长，2015年开始，它的业绩如火箭般蹿升，营收从3.13亿涨到14.62亿，净利润从区区2600万涨到4.33亿，三年复合增长率分别为67.16%和155.37%。2018年，营收和净利润增速有所放缓，但仍然超过了50%。其中的关键还是净利率的提升，而而净利润率大幅提升的主因则是因为毛利率的快速提高——从2014年的18%一路涨到2018年的45%。

毛利率指标往往能传达出很多的信息，首先一个，高毛利率的公司要么是产品具有非常高的技术含量，要么是商业模式特别牛逼，又或者是产品具有稀缺性；其次， 一家公司能在多年里维持高毛利率还能反映出行业没有恶性竞争的趋向，具有一定的壁垒。结合锐科激光的业务可知，公司所处的激光设备制造行业属于高端制造业，业务具有独特性，产品附加值高，属于高利率润率、低周转型的企业。高达45%的毛利率，说明其市场价值相对于原始制造的溢价能力很高，30%的净利润率几乎可以媲美各种消费白马。所以，接下来要观察其业务的发展是否能够保住其差异化的优势地位维持住高毛利率和高净利润率？高净利润率会不会引来竞争，同时竞争会不会导致它的利润率走低？这是第二个问题。

此外，在成本费用端，营业成本占比在2017年下降9个pct，管理费用的支出占比也下降了30%，看来搞清楚那一年发生了什么非常关键。此为第三个问题。费用端方面，锐科支出的大头是研发投入，占比将近50%，看得出来，这家公司具有技术密集的特征。其销售费用占比在2018年之前一直不超过30%，2018年这一指标达到了36%，说明公司上市融资之后，加大了销售和渠道的扩展，并且这一趋势一直延续到2019年，Q1财报显示，它的销售费用占比一下提高到了41%。对这样的企业来说，只要下游需求在，销售力度的放大或者渠道建设的投入增加，势必驱动公司业绩快速成长。还有一个值得关注的地方是，2017年，锐科管理费用占比一下子下降了3.64pct，当时发生了什么还有待于后续的研究厘清，猜测或许是公司的重头研发项目结出阶段性的成果，当然，也不排除是为了上市而提前为财报「化妆」。

上市之前，公司固定资产占比将近20%，属于重资产类型的业务，2018年上市募集资金后，这一占比降到6.88%，未来随着募投项目的展开，固定资产/在建工程占比将提升。锐科刚上市不久，目前没有有息负债，负债以应付账款为主，是假性负债。第四个问题是，接下来需要研究，其规模的扩张是否需要以较高的资本支出为前提？是否需要不断的融资进行资产再投入？

2018年，公司的应收账款占比巨幅提升（181%），总营收占比达到15%，这一现象是阶段性的还是趋势性的，有待进一步的确认。锐科维持日常营运水平所花费的资金（净营运资金）2017年之前低于2亿，之后随着营收的放大而增至当前的4亿，占总资产的23%。其经营效率在整个电子设备制造行业出类拔萃，现金周期是118天，这说明它从生产、销售再收回现金需要118天的时间，这个水平在电子设备制造业里排名前5。

ROE方面，上市前达到68%的超级水平，可见生意非常赚钱。上市后，ROE降到33.94%，从生意的特征来看，公司的财务杠杆不高，可操作的空间大。同时，2016和2017两年，总资产周转率的高点都在1以上，募投项目达产后，这块还有上升潜力。值得关注的是，公司当前营收收入只有14亿，规模不算大，但它有一个非常高的净利润率，如果其所处行业和市场空间足够大的话，那锐科激光这家公司就需要特别关注了，因为那将体现出如下的优秀特质：小公司，大行业，高利润率。

总体而言，锐科激光的财务数据是非常好的，其业务正处于高速的发展中。问题主要是以下这些：

  1. 身为制造业企业，为什么享有如此之高的净利率？ 什么导致了它净利率的高速成长？
  2. 它的高毛利率是暂时性的，还是说未来会成为常态？高净利润率会不会引来竞争，同时竞争会不会导致它的利润率走低？
  3. 2017年，营业成本大下降9个pct，同时管理费用率下降3.64pct，那一年发生了什么？
  4. 其规模的扩张是否需要以较高的资本支出为前提？是否需要不断的融资进行资产再投入？
