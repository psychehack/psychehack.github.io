锐科激光财务分析

发表于 2019-08-07 | 分类于 金融交易

显然，锐科激光正处于成长期，业务的快速扩张带来了超高速成长，2015年开始，它的业绩如火箭般蹿升，营收从3.13亿涨到14.62亿，净利润从区区2600万涨到4.33亿，三年复合增长率分别为67.16%和155.37%。2018年，营收和净利润增速有所放缓，但仍然超过了50%。在业绩快速增长的三年里，有一个现象值得关注——净利润增速高于营收增速，原因一是由于毛利率的提升，二是费用端的支出，尤其是管理费用支出占比降低了30%所致。

公司2014年的毛利率还只有18.08%，到2018年就变成了45.32%，四年时间提高了150%。较高的毛利率之下，其净利率也高达30%，这个水平在A股整个电子行业内可排进前十。由此也引发了一系列的问题——究竟是什么导致了这家公司毛利率的快速增长？ 是因为行业发生了变化？原材料大跌，成本下降？还是公司的新产品迎来了爆发？锐科激光具有什么独特的能力，又或者其产品有什么稀缺性？它有什么壁垒和护城河，可以帮助其获得竞争优势呢？

在费用端，锐科支出的大头是研发投入，占比将近50%，公司具有技术密集型企业的特征。其销售费用占比在2018年之前一直不超过30%，2018年这一指标达到了36%，说明公司上市融资之后，加大了销售和渠道的扩展，并且这一趋势一直延续到2019年，Q1财报显示，它的销售费用占比一下提高到了41%。对这样的企业来说，只要下游需求在，销售力度的放大或者渠道建设的投入增加，势必驱动公司业绩快速成长。此外，还有一个值得关注的地方是，2017年，锐科管理费用占比一下子下降了3.64pct，当时发生了什么还有待于后续的研究厘清，猜测应该是公司的重头研发项目结出成果，还有一种可能，是为了上市而提前为财报「化妆」。

上市之前，公司固定资产占比将近20%，具有典型制造业的特征，2018年上市募集资金后，这一占比降到6.88%，未来随着募投项目的展开，固定资产/在建工程占比将提升。锐科刚刚上市不久，目前没有有息负债，负债也以应付账款为主，2015年以来，应收账款的增速始终低于营收增速。

锐科的经营效率在整个电子设备制造行业出类拔萃，其现金周期是118天，这说明它从生产、销售再收回现金需要118天的时间，这个水平在电子设备制造业里排名前列。公司维持日常营运水平所花费的资金（净营运资金）2017年之前低于2亿，之后随着营收的放大而增至当前的5亿，占总资产的21%。

锐科所处的激光设备制造行业属于高端制造业，业务具有独特性，产品附加值高，属于高利率润、低周转型的企业。公司高达45%的毛利率，说明其市场价值相对于原始制造的溢价能力很高，30%的净利润率几乎可以媲美白酒企业。所以，接下来要观察其业务的发展是否能够保住其差异化的优势地位，也就是维持住高毛利率和高净利润率？这种高毛利率是暂时性的，还是说未来会成为常态？高净利润率会不会引来竞争，同时竞争会不会导致它的利润率走低？

ROE方面，公司上市前的总资产周转率一度达到了1.43的优秀水平，随着启动上市融资，其总资产的周转率能否得到提高是未来需要重点观察的地方。锐科正处于ROE的扩张阶段，如果下游需求旺盛，公司可以实现更高的营业规模，它又不需要在扩张中频繁的融资和进行资本性支出，只需要聪明地利用一点儿杠杆资金，就可以达到优秀的ROE水平。
